🐦 Water-redstart: the chirping viewer v2.1 (双平台版) - 极速鸟类照片挑选小助手

🪟 Windows 使用说明

本程序为封装好的单文件应用程序，无需安装任何环境，直接将 Water-redstart_v2.1.exe 复制或移动到你的照片文件夹内，双击运行即可开始挑图！

🍎 macOS 小白一键运行指南

由于本程序现已全面支持 Windows / macOS 双平台运行，对于不熟悉终端操作的 macOS 电脑小白用户，您可以通过以下方式轻松制作一个“双击即用”的脚本：

  1. 在本程序所在文件夹中，新建一个纯文本文件，命名为 启动看鸟.command（注意后缀名是 .command）。
  2. 使用普通的文本编辑器（如 Mac 自带的文本编辑）将其打开，将以下代码完整复制并粘贴进去，然后保存：
#!/bin/bash
cd "$(dirname "$0")"
pip3 install pillow rawpy
python3 photo_viewer.py
  1. 首次权限激活（仅需一次）：
    打开 Mac 的“终端（Terminal）”，输入 chmod +x （注意 +x 后面必须保留一个空格），然后直接将刚才创建的 启动看鸟.command 文件用鼠标拖到终端窗口里，按下回车即可。

完成以上步骤后，以后每次在 Mac 上看图只需 双击 启动看鸟.command 就能直接打开软件啦！

✨ 软件简介

Water-redstart 是专为摄影师、观鸟记录者以及需要从海量照片中快速筛选出图的用户打造的一款本地极速看图工具。

告别缓慢的文件夹拖拽和繁琐的复制粘贴操作，通过 Water-redstart: the chirping viewer，你只需使用键盘热键进行“快准狠”的二元判断（跳过或挑出），即可用极高效率梳理成百上千张单反照片。它被设计得轻量、活泼且极度流畅。

🚀 核心功能亮点

1. 极致交互：全键盘沉浸式筛选

  • [D] 下一张：这张一般般，跳过！
  • [W] 挑出：这张很棒！一键触发清脆的“叽叽”鸟鸣音效，自动将当前源文件安全复制到选中的独立子文件夹，毫不拖泥带水。
  • [S] 复制到剪贴板：看到一张想分享的高光时刻？直接将源文件路径放入系统剪切板（彻底解决大分辨率照片复制后提及膨胀变数百兆的问题），随时完美无损地粘贴到微信或文档中。
  • [A] 自动撤销：手滑了？别着急，按下撤销键，自动退回上一张。如果是“挑出”手滑，程序会自动跟进帮你把刚才放进子文件夹里的废片删干净（放飞小鸟）。
  • 顶端灵动进度条：上方自带可视化进度条，使用鼠标拖动它就可以在全盘大跨度照片中进行快进和翻页！
  • 灵活的倒序审图：开启设置里的“倒序选片”，系统将一秒自动头尾翻转，方便您顺着刚刚拍摄完成的最新照片一直往前看！
  • 以上所有热键均可随时进入设置面板修改为您习惯的顺手按键，哪怕是方向键也同样支持哟！

2. ⚡ RAW/巨型原图极速翻页体验

告别每翻一张 30MB 的原图就要卡顿转圈的噩梦！

  • 全面支持更多 RAW 格式：本版本新增支持 .naw 与 .nrw 格式文件，满足更多型号相机的看图需求。
  • 重构的底层极速通道：通过拦截并使用底层 C 库在图片读取阶段完成分辨率的降采样解码，使得巨型文件占用内存减半且画质锐利不降，体验飞升。
  • 后台多线程深度预读：它像个聪明的小管家，在你看眼下这张时，它在后台已将后多达五十张照片悄悄放入内存缓存。每一次按键翻页都是“秒切”，哪怕以“一拍三张”的节奏盲开连按，也绝不会有一丝卡顿。

3. 🖱️ 沉浸式拖拽放大

  • 鼠标滚轮局部缩放：看鸟当然要数毛！使用鼠标滚轮可随心所欲悬停放大照片局部的细节。
  • 左键平滑拖拽：放大后支持鼠标左键按住拖动自由平移，享受顺滑细节审视体验！

4. 📂 智控文件夹与善后清理

  • 智能记忆读图进度：不想下次从大分类的第一张重看？不慌！软件带有每次随用随保存的功能，下一次打开它会在上次结束的地方等着您。
  • 贴心多功能结尾向导：当你梳理完该目录下最后一张照片，程序会立刻为你颁发“小红花”，并直接弹出快捷处理面板：你可以选择一键无痕抹除已经挑废了的整个文件夹原片，甚至还可以让老版本看图程序上演“大义灭亲”——顺手把自己也删除得干干净净。

🎨 外观与个性化

  • UI 色彩可调：觉得默认的橘粉色和淡黄色底色不搭当前心情？进入魔法设置，您可以使用调色盘将整个信息横幅任意改色！
  • 界面呼吸感优化：顶部设置了 UI 字体缩放自适应尺寸控制拉杆，全屏观图时（按 F11）它不强夺眼球，确保鸟儿的美丽羽毛主宰屏幕。
  • 红尾水鸲图标：拥有一枚萌动可爱的手绘红尾水鸲（蓝身子、黑嘴巴、橙色小尾巴）形象的专属图标，桌面存在感高。

📥 如何使用

将编译好的独立 Water-redstart_v2.1.exe 放置在存放杂乱待选照片的随便哪个“图库夹”旁边，双击运行即可。

环境依赖：
由于已封装为单文件应用程序（Exe），无需任何 Python 前置插件或繁重的看图器支持，开箱即用。
